# Atari Game
Repository for Atari 2600 game.

## Possible Game Names

The Knsh. Kid

The Knsh. Kid: R. vs. BLM

The Knsh. Kid: R. vs. Antifa

K. R.

R.

St. R.

St. R. of K.

## Game Description
Atari 2600 game based on K. R. where the player is R. who has to save Knsh. by shooting looters and arsonists.
